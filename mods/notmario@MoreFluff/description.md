# More Fluff

A mod that adds:
- 45 jokers
- 16 Colour Cards
- 22 45 Degree Rotated Tarot Cards

## Installation
- Requires [Steamodded](https://github.com/Steamopollys/Steamodded/) v1.0.0 

## Extra content
 - [CardSleeves](https://github.com/larswijn/CardSleeves): Decks can be used as card sleeves
